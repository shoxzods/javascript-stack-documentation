// ------------------------------------- PostgreSql ----------------------------------------- //

PostgreSql - это open-source , беспоатный и релиационный система датабазы.

PostgreSql подердивает и релиационную ( sql ) и нереалиционную датубазу ( JSON queries ).

PostgreSql это бекенд датабаза для динамических вебсайтов и веб приложений.

PostgreSql подерживает для оченных важный языков программирования:
    python , java , nodejs , go , ruby , c / c++ , c# , ruby.
